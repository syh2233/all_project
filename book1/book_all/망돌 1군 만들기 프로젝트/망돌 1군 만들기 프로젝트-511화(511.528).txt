망돌 1군 만들기 프로젝트 511화
“제가 점심 식사 때 계속 먹을 것을 권했는데, 혹시 그것 때문에…….”
“그건 아닐 것 같은데요. 어젯밤에 숙소에 들어오셨을 때부터 컨디션이 좋지 않아 보이셨거든요.”
안절부절못하는 차윤재의 말을 끊은 한수현은 성해온이 했던 말을 떠올렸다.
- 그렇게 걱정할 필요는 없다. 그냥 한숨 자고 나면 괜찮아질 정도야.
- ……!
- 너희가 그렇게 귀가 따갑게 말해오던 아플 때 쉬는 걸 실천해 보는 거다.
“해온 형이 그렇게 말하셨으니, 저희는 스케줄에 최선을 다하면 되는 겁니다.”
“수현이 말이 맞아.”
곧바로 긍정한 류인이 걱정으로 흐려진 얼굴의 멤버들의 어깨를 다독였다.
“해온이는 푹 쉬고 있을 테니까, 걱정은 잠깐 멈추고 라디오 준비하자.”
“듣다 보니 수현이와 형님의 말씀이 옳습니다. 분명 쉬고 계실 테니, 저는 최선을 다해 녹화에 임해야겠습니다! 해온 형님 몫까지요!”
“잘 생각했어.”
작게 웃은 류인이 ‘아’ 소리를 낸 것도 그때였다.
“나는 잠깐 작가님한테 다녀올게. 아까 전달 사항이 있다고 하셨거든.”
원래라면 항상 성해온이 하던 일이다.
자신이 하겠다해도 고개를 저으면서 해내던…….
“음.”
라디오 방송국 복도를 걷던 류인이 침음을 흘리며 혼잣말했다.
“해온이 없이 참여한 스케줄이 처음도 아닌데…… 매번 이렇게 빈자리가 크단 말이야.”
정말 그랬다.
멤버들 앞에선 의연한 척을 했지만, 성해온이 없을 때 빈자리를 느끼는 건 자신도 마찬가지였다.
“그만큼 해온이가 의지가 됐다는 거겠지…….”
솔직히 조금 부끄러운 일이었다.
따지자면, 자신도 성해온과 함께 이 그룹의 연장자였으니 말이다.
“무슨 생각을 그렇게 열심히 해요?”
“……!”
“으하하, 놀랐어요?”
“조금…… 언제부터 따라왔어? 기척을 못 느꼈는데.”
“오호라, 이거 의외의 재능을 찾은 것 같은데?”
류인의 어깨에 팔을 걸친 최승하가 물었다.
“그나저나, 형도 생각이 많아 보이네요.”
“으음, 생각이라기보단…….”
“의외여서 놀랍죠.”
정곡을 찌른 말에 류인의 입이 다물렸고, 최승하는 그럴 줄 알았다는 듯이 눈을 접어 내렸다.
“그렇게 자발적으로 스케줄을 캔슬한 건 처음이잖아요.”
“……!”
“본인의 상태가 어떻건 스케줄을 강행하려던 사람인데…….”
순간적으로 서늘해진 분위기에 류인이 멈칫한 순간이었다.
“드디어 우리의 마음을 알아주는 건가?”
최승하가 장난스럽게 덧붙인 것이다.
“오늘 아주 잘해 버려서 다음에도 아플 땐 쉬어도 된다는 생각을 각인시켜 줘야겠어요!”
찰나에 느꼈던 분위기는 착각이었다는 것처럼, 최승하가 헤실 웃었다.
“흐음, 이거 의욕이 상당히 불타는데?”
눈을 천천히 깜빡이던 류인이 이내 웃음을 터뜨렸다.
“좋아, 우리 파이팅하자.”
“그럼요!”
역시, 평소의 최승하와 다를 바 없었다.
내가 무슨 생각을 한 거람.
* * *
몸을 둥글게 만 나는 툭툭 끊기는 신음을 토해냈다.
‘기억’을 되찾은 건, 이번이 처음이 아니다.
나는 스스로 지워냈던…… 어린 ‘이해온’의 기억을 되찾은 적이 있으니까.
하지만 말이다.
그때와 지금은 완전히 달랐다.
어째서?
‘내’ 기억이 아닌, ‘타인’의 기억이어서?
[SYSTEM ERROR CODE : 520 UNKNOWN ERROR!]
‘이해온’의 기억을 되찾을 땐, 매번 이 메시지가 떠올랐다.
그러나, ‘성해온’의 기억이 밀려들어 오고 있는 지금은…….
[SYSTEM ERROR CODE : 403 FORBIDDEN ERROR!]
그때와 전혀 다른 메시지가 떠올랐다.
직역하자면, 금지된 오류라는 의미의 메시지.
명확히 알 수는 없지만, 본능적으로 느껴졌다.
누군가가 내게 ‘성해온’의 기억을 ‘강제’로 주입하려 하고 있다는 것.
문제가 있다면…….
그 기억이 내게 허락되지 않은 종류의 기억이라는 거겠지.
[system] 인간! 인간! 우리 메시지가 보여? 지금 무슨 일이 벌어지고 있는지 설명해 줄 수 있어?
[system] 우리는 인간, 네가 보이지 않아! 시스템에 오류가 생겼어!
흐릿한 시야 속으로 시스템 관리자들의 메시지가 보였다.
그럼에도…….
나는 대답조차 할 수 없었다.
[ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR]
더는 버틸 수가 없었기 때문에.
그래.
이제는 나 역시도 한계였다.
* * *
──지직, 지지직
공간을 가득 채운 모니터가 동시다발적으로 꺼지거나, 지지직거리고 있었다.
그리고 그 사이에 놓인 시스템 관리자들은 그대로 얼어붙었다.
[……인간에게 무슨 일이 생긴 거지?]
[아무것도 보이지 않아!]
이들은 시스템에 오류가 생기자마자 성해온에게 연락을 시도했지만…… 조금도 닿지 않았다.
말 그대로 먹통.
[이, 인간이 우리가 보낸 메시지에 답하지 않았어! 정말 무슨 일이 생겼나 봐!]
[침착해! 메시지가 제대로 가지 않았을 수도 있잖아!]
당장에라도 터질 것 같은 눈물을 꾹 참고 차분하게 말을 꺼낸 시스템 관리자가 먹통이 된 시스템을 매만졌다.
[바보야! 그래봤자 되는 건 없어! 이렇게 시스템에 오류가 생겼을 때는 그저 기다리는 게 정답이라는 걸 알잖아!]
틀린 말이 아니었다.
이렇게 시스템에 의문의 오류가 생겼을 땐, 시간이 곧 정답이었다.
이들의 능력으로 해결되는 종류가 아니었으니까.
[그래도…… 그냥 기다릴 수가 없는 걸.]
시스템 관리자의 시선이 허공으로 향했다.
잔뜩 깨진 화면 속에는 성해온에 대해 기록된 문서.
그러니까…….
성해온에게는 ‘상태창’이란 형태로 요약되어 띄워지는 문서가 담긴 화면이 어렴풋이 보였다.
[저길 봐.]
깨져서 제대로 보이지 않는 글자 속으로, 선명하게 보이는 게 하나 있었다.
원래라면 스탯이 쓰여 있어야 할 정신력이 에러 처리되어 있었다.
[인간의 정신력이 완전히 무너져 내렸잖아.]
[…….]
[자세히는 몰라도 인간이 받아들일 수 없는 무언가가 터져 버린 거야.]
시스템 관리자의 목소리가 점점 작아졌다.
[지금 인간은 절벽에 서 있는 것과 다름이 없어…….]
* * *
무의식 속에서 누군가의 목소리가 점차 선명해졌다.
그러니까…….
멤버들의 목소리였다.
천천히 눈꺼풀을 떠올리자, 내가 눈을 뜬 걸 본 차윤재의 동공이 확장됐다.
“……! 일어나셨, 아, 아니, 그게 중요한 게 아니라…… 몸은 괜찮으십니까?”
그리고 나는 벽에 달린 시계를 눈에 담았다.
불행 중 다행으로, 오래 정신을 잃은 게 아니었다.
이 시간이면, 이 녀석들도 스케줄을 마치고 숙소에 온 지 기껏해야 1시간 안팎일 터…….
충분히 수습할 수 있었다.
“너희 들어오는 소리도 못 듣고 잤나 본데.”
자연스럽게 운을 뗀 내가 상체를 일으킨 순간이었다.
한수현이 내 행동을 저지한 것이다.
“잠시…….”
열을 재는 것처럼, 다른 한손을 본인의 이마에 올린 한수현이 심각한 얼굴을 했다.
“열은 없는 것 같습니다만…… 그래도 휴식이 더 필요할 것 같습니다. 저희가 숙소에 들어왔을 때, 식은땀을 흘리며 앓고 계셨습니다. 깨워도 반응이 없으셨고요.”
“…….”
“더 늦게 일어나셨다면, 아마도 병원으로 향했을…….”
“몸 상태가 좋지 않았던 건 맞아. 그러니 내 발로 스케줄을 빠진 거고.”
“해온 형.”
“그런데 이제는 정말 괜찮아. 오히려 한숨 자고 나니 개운한데.”
내가 눈을 뜨자마자 한 일은, 바로 상태 체크다.
식은땀을 제외하면 발열이나 출혈 등 겉으로 보이는 증상은 존재하지 않았다.
아까보다 나아졌을 뿐, 두통은 여전히 존재했으나…….
이건 티를 내지 않으면 그만이었다.
“만약 괜찮지 않았다면, 내가 먼저 다른 스케줄도 빠져야겠다고 말했을 거다.”
이마에 올려진 한수현의 손을 떼어낸 나는 멤버들과 시선을 마주했다.
할 말이 많아 보이는 얼굴들이지만, 녀석들은 별다른 말을 꺼내지 못했다.
말마따나 겉으로 보이는 커다란 이상이 없으니, 무어라 강제할 수도 없는 것이다.
나 역시도 그걸 알기에 말을 꺼낸 거고.
“내일은 오전 스케줄도 없겠다, 더 쉴 수 있으니 걱정할 필요 없어.”
저런 얼굴들을 보니 양심이 찔리기야 한다만…… 어쩔 수 없는 일이다.
무엇보다.
내 몸뚱아리 자체가 많은 걸 배려할 수 있을 만한 상태가 아니거든.
“나는 씻고 나올 테니까, 다들 들어가 쉬어라.”
한 음절, 한 음절.
이상이 느껴지지 않게끔 말을 마친 나는 욕실 안으로 들어갔다.
──타악!
그리고 문이 닫히기 무섭게, 나는 그대로 미끄러져 주저앉았다.
실은, 정신을 차리자마자 알 수 있었다.
……지금 내가 정상적인 상태가 아니라는 것을.
[ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR]
어찌 보면 당연한 일이었다.
눈을 뜨자마자, 익숙한 에러메시지가 계속해서 떠올랐으니까.
‘정신이…….’
마른 얼굴을 쓸어내린 나는 속으로 되뇌였다.
‘정신이 제대로 차려지지가 않아.’
이상한 기분이었다.
평소보다도 더 울렁였고, 평소보다도 더 토악질이 나올 것 같았다.
마치, 무언가가 잘못된 것처럼.
그리고 나는 얼마안가 그 이유를 알 수 있었다.
내 예상이 맞았거든.
허공에 떠오른 상태창을 보던 나는 미간을 찌푸렸다.
‘불길한 직감은 어째서 달라지지 않는 건지 모르겠군.’
그래, 정신력이 무너져 내렸다.
‘추측컨대, 내가 받아들이기 버거운 것이었던 거겠지.’
도저히 견딜 수 없을 정도의 충격이 가해졌던 걸 생각하면…… 내 정신력이 일시적으로 무너져 내린 게 이상하진 않았다.
문제는…….
내 몸뚱아리 자체가 시한폭탄이 됐다는 거겠지.
버티기 힘든 일이 있을 때마다 나름대로 버틸 수 있었던 데에는…… 정신력이 커다란 비중을 차지했다.
그리고 지금은 그게 사라진 상태.
비유하자면, 강물을 막아줄 댐이 무너진 것이나 다름없었다.
‘정말이지, 끝내주게 답이 없는 상황인데.’
두통을 삼켜낸 나는 보기만 해도 어지로운 에러 메시지를 눈에 담았다.
[ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR ERROR]
“받아들여야 해.”
가장 급선무는 바로 이거였다.
내게 이 페널티를 준 의문의 존재의 뜻대로…….
이 빌어먹을 기억을 받아들이는 것.
* * *
모두가 잠든 새벽.
성해온의 기억을 받아들이던 나는 숨을 힘겹게 몰아쉬며 눈을 크게 떴다.
여태껏 성해온의 기억이 궁금했던 적이 한번도 없었다면 거짓말이다.
하지만 말이다.
성해온의 감정의 형태가 이러리라고는…….
나는 조금도 예상하지 못했다.
