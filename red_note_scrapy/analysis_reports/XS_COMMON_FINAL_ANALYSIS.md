#### 3. JavaScript代码分析
从vendor-dynamic.77f9fe85.js分析发现：

- `p.lz` = `encodeUtf8` - UTF-8编码
- `p.xE` = `b64Encode` - Base64编码
- `y` = 包含时间戳、平台、设备类型、URL哈希等参数的对象
